Built with an Accessibility First and Mobile First mindset, adherence to Web Content Accessibility Guidelines (WCAG) was a requirement. To learn more about the considerations taken for accessibility, please view the Figma File.

The Hardcover.app GraphQL API is being utilized to retrieve my Currently Reading and Recently Read books through the usage of Netlify Functions. This ensures instantaneous synchronization between my portfolio and the hardcover.app in real time.
